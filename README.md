# Flutter Custom ThemeData

Um simples projeto mostrando como criar extensões personalizadas para ThemeData.

- Versão: Flutter 3.35.3

### Run

1. Clone o repositório
2. Execute `flutter pub get`
3. Execute `flutter run`
